# GTM Vue Project

這是一個 Vue.js 專案，整合了 Google Tag Manager (GTM) 功能。

## 🚀 線上預覽

網站部署在：https://larryz1215.github.io/gtm-vue/

## 📁 專案結構

- `vue-gtm/` - 主要的 Vue.js 應用程式
- `.github/workflows/` - GitHub Actions 自動部署配置

## 🛠️ 本地開發

```bash
cd vue-gtm
npm install
npm run dev
```

## 📦 建置

```bash
cd vue-gtm
npm run build
```

部署會透過 GitHub Actions 自動進行。
